# 📹 CCTV Knowledge Base

A practical repository for learning, documenting, and sharing knowledge about Closed-Circuit Television (CCTV) systems.
This repo contains real-world experience, installation guides, troubleshooting methods, networking concepts, and security best practices related to CCTV technologies.

---

# 📚 What You Will Find Here

* CCTV fundamentals
* Camera types and comparisons
* DVR / NVR configuration
* IP camera setup
* Networking for CCTV systems
* Remote access configuration
* Storage and recording management
* Troubleshooting techniques
* Cable management and installation tips
* Security best practices
* Real-world deployment experiences
* Maintenance procedures
